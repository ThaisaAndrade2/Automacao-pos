### 1. Instale as dependências e os navegadores do Playwright

Execute o seguinte comando:

```bash
python -m pip install -r requirements.txt
playwright install´´´


### Configuração da senha (arquivo secrets.py)
Para manter sua senha protegida, usamos um arquivo chamado `my_secrets.py`, que **não está no repositório** por segurança.

1. Copie o arquivo de exemplo:
   ```bash
   cp my_secrets_example.py secrets.py´´´
